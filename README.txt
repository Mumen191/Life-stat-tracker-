LIFE RPG — WORKING BUILD

Upload these six files to the root of your existing GitHub repository:
index.html
styles.css
app.js
manifest.json
service-worker.js
README.txt

Then commit directly to main. GitHub Pages will update automatically.

To refresh the iPhone app:
1. Delete the old Home Screen icon.
2. Go to Settings > Safari > Advanced > Website Data.
3. Search github.io and remove the old site data.
4. Open your GitHub Pages URL in Safari.
5. Share > Add to Home Screen.

All personal data is saved only in the browser on that device.
